# Assignment 1 — Advanced Programming (Go)

## Description
This project was developed as part of the Advanced Programming 1 course.
It demonstrates core Go concepts including packages, structs, methods,
interfaces, maps, and slices.

- employee — employee salary calculation using interfaces
